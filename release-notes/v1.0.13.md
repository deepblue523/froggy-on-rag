## Summary

**Froggy RAG MCP** is a self-contained desktop app that ingests your documents into a local vector store, runs semantic search, and exposes the same capabilities over **MCP** (stdio + REST) so tools like Claude Desktop can use your corpus.

## Highlights

- **Local RAG pipeline** — ingest, chunk, index, and search without sending your files to a hosted “knowledge base” product.
- **Supported formats** — Word (`.docx`), Excel (`.xlsx`), PDF, CSV, and plain text.
- **Queue-based ingestion** — batch adds, status tracking, optional directory watching (including recursive watch).
- **Semantic search** — query the vector store from the UI; inspect chunks and metadata.
- **MCP server** — stdio for MCP clients, REST for HTTP callers, plus a **CLI** for tools/search/stats (`froggy-rag-mcp` / `mcp-cli.js`).
- **Desktop UX** — Electron UI with ingestion, vector store, search, and server sections; settings and layout persist.

## Install

- **Windows:** use the **`.exe` (NSIS)** installer from this release for the best match with in-app updates; an **MSI** is also included if you prefer it for deployment.
- After install, data lives under **`%USERPROFILE%\froggy-rag-mcp\data`** (vector DB, settings, watch config).
